VECTORS vs RASTERS
the Ultimate Match!

En la ultima version de HTML encontramos dos nuevos elementos, estos son las etiquetas <figure> y <figcaption> que corresponden a un elemento grafico y su definicion. Si bien, todavia seguimos utilizando la etiqueta <img> el uso de esta nueva etiqueta refuerza la estructura semantica de nuestro documento HTML.

Uno de los aspectos mas importantes de la implementacion de CSS junto a nuestros documentos HTML, corresponde al uso estilizado de imagenes. Entre los formatos mas utilzados para Web, encontramos dos estandares MAPA DE BITS (Raster) y GRAFICOS GENERADOS POR COMPUTADORA (vector)

MAPA DE BITS: es una estructura o fichero de datos que representa una rejilla rectangular de píxeles o puntos de color, denominada matriz, que se puede visualizar en un monitor, papel u otro dispositivo de representación.
Entre los formatos mas utilizados para Web encontramos los siguientes
JPG (Joint Photographic Expert Group)
PNG (Portable Network Graphics)
GIF (Graphics Interchange Format)

GRAFICOS GENERADOS POR COMPUTADORA: imagen digital formada por objetos geométricos dependientes (segmentos, polígonos, arcos, muros, etc.), cada uno de ellos definido por atributos matemáticos de forma, de posición, etc. Por ejemplo un círculo de color rojo quedaría definido por la posición de su centro, su radio, el grosor de línea y su color.
Entre los formatos mas utilizados para Web encontramos los siguientes
SVG (Scalable Vector Graphics)
VML (Vector Markup Language)
SWF (Small Web Format)

Si bien podemos utilizar, como valor del atributo SRC, la direccion URL de algun repositorio de imagenes, es aconsejable que dispongamos de las imagenes que utilizaremos para nuestro sitio y las almacenemos en una carpeta donde se encuentre nuestro proyecto. Esto se debe a que al utilizar una fuente externa, estas imagenes pueden estar dispuestas a modificacion, ya sea que se modifique la direccion URL de la misma y por lo tanto quede inaccesible o porque la misma haya sido eliminada del servidor en el que esta se encontraba almacenada.

Ejemplo de Codigo HTML

<!DOCTYPE html>

<html>

<head>

<title>Titulo de Mi Pagina </title>

</head>

<body>

<img src="Ruta de imagen" alt="texto alternativo (en caso de no cargarse)" title="titulo (no visible)">

Ejemplo de URL:
<!-- ruta absoluta (externa)-->
"https://alumni.education/assets/img/svg/alumni-logo.svg"
<!-- ruta relativa (interna)-->
"img/logo.png"

</body>

</html>
